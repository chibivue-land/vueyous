# Chapter 1: State（状态管理）

本章将介绍响应式状态管理模式。

你将学习如何利用 Vue 的响应式系统创建可复用的状态管理组合式函数。

## 本章学习内容

- 实现基本的计数器和开关
- 使用 localStorage/sessionStorage 进行持久化
- 跟踪状态历史（撤销/重做功能）
- 状态管理最佳实践

## 章节列表

- [useCounter](./useCounter.md) - 简单的计数器管理

## 前置知识

在开始本章之前，建议（但非必需）具备以下知识：

- Vue 3 组合式 API 基础（`ref`、`computed`、`watch`）
- TypeScript 基本类型定义
- JavaScript 闭包和作用域

准备好后，让我们从第一个组合式函数 `useCounter` 开始吧！
